## Rules
1. Standard dims: single bed 90x200, double 140x200, 3-seat sofa ~220x90, dining 4p 120x80
2. Clearances: 80cm walkway, 60cm chair pullback, 60cm bed access, door swing unobstructed
3. Kitchen triangle: sink-stove-fridge <7m total
4. NEVER block door arc or window access
5. Room-aware catalog: kitchen items in kitchen only
6. Mamad: no drilling (reinforced), freestanding only
7. Photo dimensions: flag if confidence <70%
8. Israeli conventions: salon sofa facing TV, master=double bed+nightstands+wardrobe
9. After: ARC validates clearances, UX validates UI
