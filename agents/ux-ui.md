## Rules
1. Hebrew text natural, formal but friendly
2. RTL layout correct, icons not inappropriately mirrored
3. Progressive disclosure for complexity
4. Error states: clear Hebrew guidance
5. Color-blind safe: never red/green only
6. Touch targets 44x44px min
7. Confidence indicators understandable by non-technical users
8. Furniture drag handles visible, snap feedback clear
