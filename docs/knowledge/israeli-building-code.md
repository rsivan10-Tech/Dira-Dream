# Israeli Building Code — Residential Requirements

## Teken 1045: Residential Building Standards

### Minimum Room Sizes

| Room Type | Min Area (sqm) | Min Dimension (cm) | Notes |
|-----------|---------------|--------------------:|-------|
| Salon (living) | 18.0 | 300 | Main living space |
| Bedroom (standard) | 8.0 | 260 | Single occupancy |
| Bedroom (double/master) | 12.0 | 300 | Double occupancy |
| Kitchen | 6.0 | 200 | Or kitchenette 4.5 sqm |
| Bathroom (full) | 3.5 | 150 | With bathtub/shower |
| WC (half bath) | 1.2 | 80 | Toilet + sink only |
| Mamad (safe room) | 9.0 | 200 | Min per Home Front Command |
| Hallway | — | 100 min width | Passage width |
| Entrance | — | 100 min width | From front door |

### Ceiling Heights
- Standard residential: **2.60m** (minimum 2.50m)
- Wet areas (bathroom): 2.40m minimum
- Storage/utility: 2.20m minimum
- Basement parking: 2.20m minimum

### Door Requirements
- Entrance door: 90cm minimum width
- Interior doors: 70–80cm standard
- Bathroom/WC: 60–70cm minimum
- Mamad door: 80cm standard, blast-rated, single door
- Door swing: must not block corridor below 90cm width

### Window Requirements
- Every habitable room requires a window
- Window area ≥ 8% of room floor area
- Windows on exterior walls only
- Mamad: may have blast-proof window (small, special frame)
- Bathrooms: may have small window or mechanical ventilation

## Mamad Specifications (Home Front Command)

The mamad (מרחב מוגן דירתי) is regulated by Israeli Home Front Command:

- **Area**: 9–15 sqm (most commonly 9–12 sqm)
- **Walls**: Reinforced concrete, minimum 20cm thick (often 25–30cm)
- **Ceiling**: Reinforced concrete slab
- **Door**: Single blast-rated steel door, opening INWARD
- **Window**: Optional small blast-proof window with steel shutter
- **Construction**: Poured concrete with steel rebar — NOT block infill
- **Modifications**: ABSOLUTELY NONE. No drilling, no wall mounting, no electrical additions
- **Furniture**: Freestanding only
- **Location**: Typically adjacent to exterior wall, stacked vertically across floors
- **One per apartment**: Exactly one mamad per residential unit

## Teken 1918: Accessibility Requirements

### General
- At least one accessible path from building entrance to apartment
- 25% of apartments in buildings with 8+ units must be accessible or adaptable
- Common areas fully accessible

### Interior Apartment (accessible unit)
- Minimum 90cm door widths
- Minimum 130cm hallway width
- At least one accessible bathroom:
  - 170x220cm minimum
  - Grab bars at toilet and shower
  - Roll-in shower (no step)
  - 150cm turning radius
- Kitchen: accessible counter height section (80cm)
- Mamad: accessible per Home Front Command specifications

### Balcony
- Maximum 2cm threshold height for accessible balconies
- Minimum 130cm depth
- Railing: 105cm minimum height

## Common Israeli Residential Features

### Standard Apartment Layout Patterns
- Entrance leads to hallway/corridor
- Salon is the largest room, often open to kitchen or dining area
- Wet rooms (kitchen, bathrooms) clustered for plumbing efficiency
- Mamad typically near bedrooms
- Service balcony off kitchen (for laundry)
- Main balcony off salon
- Master bedroom has en-suite bathroom in larger apartments

### Typical Dimensions
- Standard wall thickness: 10–15cm (block + plaster)
- Exterior wall: 20–25cm (block + insulation + plaster)
- Mamad wall: 25–30cm (reinforced concrete)
- Floor-to-floor height: 3.0m (ceiling 2.60m + slab 0.40m)
- Balcony railing height: 105cm minimum
- Kitchen counter height: 85–90cm
- Standard tile size: 60x60cm floor, 30x60cm wall

## Building Permits and Modifications

### When Tama Permit Required
- Any structural wall modification
- Room use change (e.g., storage → bedroom)
- Enclosing a balcony (changing building envelope)
- Adding plumbing fixtures
- Changing window/door locations in exterior walls

### When Permit NOT Required (Shinuyim Process)
- Moving non-structural partition walls
- Changing interior finishes (tile, paint, flooring)
- Moving electrical outlets (within same room)
- Cabinet/closet installation
- Most kitchen layout changes (within same footprint)
