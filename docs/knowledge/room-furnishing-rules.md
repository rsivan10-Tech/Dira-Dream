## Room-Type Validation

Furniture catalog items must declare which room types they belong to:

```python
ROOM_FURNITURE_MAP = {
    'salon': ['sofa', 'coffee_table', 'tv_stand', 'dining_table',
              'dining_chair', 'armchair', 'bookshelf', 'side_table'],
    'bedroom': ['single_bed', 'nightstand', 'wardrobe', 'desk',
                'chair', 'bookshelf', 'dresser'],
    'master_bedroom': ['double_bed', 'nightstand', 'wardrobe', 'dresser',
                       'armchair', 'mirror', 'ottoman'],
    'kitchen': ['counter', 'sink', 'stove', 'refrigerator', 'oven',
                'dishwasher', 'microwave', 'island'],
    'bathroom': ['toilet', 'shower', 'bathtub', 'vanity',
                 'washing_machine', 'dryer'],
    'mamad': ['desk', 'chair', 'bookshelf', 'single_bed',
              'nightstand', 'wardrobe_small', 'crib'],
    'balcony': ['outdoor_table', 'outdoor_chair', 'planter', 'lounge_chair'],
    'service_balcony': ['washing_machine', 'dryer', 'utility_shelf', 'drying_rack'],
    'entrance': ['shoe_cabinet', 'console_table', 'mirror', 'coat_hook'],
    'hallway': ['console_table', 'bookshelf_narrow'],
    'storage': ['shelf_unit'],
}

def validate_room_furniture(room_type: str, item_category: str) -> bool:
    """Check if furniture item belongs in this room type."""
    return item_category in ROOM_FURNITURE_MAP.get(room_type, [])
```
